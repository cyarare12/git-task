Class Diagram for Task Manager Application

Classes:

Task (Model)
- Attributes:
  - id: int (private)
  - title: string (private)
  - description: string (private)
  - status: string (private)  // e.g., "pending", "completed"
  - due_date: date (private)
- Methods:
  + __init__(id, title, description, status, due_date): void
  + get_details(): dict
  + update(title, description, status, due_date): void

TaskController
- Attributes:
  - model: Task (private)
  - view: TaskView (private)
- Methods:
  + create_task(data): void
  + read_task(id): Task
  + update_task(id, data): void
  + delete_task(id): void
  + list_tasks(): list

TaskView
- Attributes: None
- Methods:
  + display_tasks(tasks): void
  + show_form(): void
  + show_message(message): void

Relationships:
- TaskController --uses--> Task (association)
- TaskController --uses--> TaskView (association)
