Relationships:
- NoteListView --> Note : uses
- NoteDetailView --> Note : uses
- NoteCreateView --> Note : creates
- NoteCreateView --> NoteForm : uses
- NoteUpdateView --> Note : updates
- NoteUpdateView --> NoteForm : uses
- NoteDeleteView --> Note : deletes

All views inherit from Django's generic views (ListView, DetailView, CreateView, UpdateView, DeleteView)
